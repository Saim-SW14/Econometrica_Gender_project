# Econometrica_Gender_project
Были использованы данные с сайта Российского мониторинга экономического положения и здоровья населения НИУ ВШЭ, данные для репрезентативной выборки индивидов: https://www.hse.ru/rlms/spss \
Исследовали два города: Казань и Владивосток в 2010 и 2017 годах \
В качестве задачи была поставлена цель изучить влияние пола на заработную плату, выбрать контрольные переменные, найти возможные причины смещенности оценок и проверить соотвествие предпосылкам теоремы Гаусса-Маркова и по итогу выбрать лучшую модель
